# Tests Directory

This directory contains all testing scripts and test runners for the Movie Recommendation API.

## Files:
- `api_test_suite.py` - Main API test suite
- `live_api_test.py` - Live API testing script
- `run_tests.py` - Test runner script
- `test_api.bat` - Windows batch script for running tests
- `test_auth_live.py` - Live authentication testing
- `test_db_connection.py` - Database connection testing
- `test_deployment.py` - Deployment testing script

## Usage:
- Run `run_tests.py` for comprehensive API testing
- Use `test_api.bat` for Windows environments
- Individual test files can be run separately for specific testing needs
